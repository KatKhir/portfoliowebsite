Mangler titel

Nu skal det handle om dynamiske websites. I dette forløb slog vi Javascript sammen med css animationer til at udvikle et spil. Vi blev introduceret for Adobe illustrater hvori vi skulle arbejde med formgivning af grafiske spil-elementer. Vi skulle bruge nye idégenereringsteknikker, lave figur-skitser, lære om css-positionering og om export af figur til svg. Derudover skulle vi fremstille diagrammer over vores spil-forløb og kode ud fra dem. Vi fik gode introduktioner til hvordan Javascript fungerer, herunder funktioner, metoder (addClass/removeClass), events (load, click, animationend, animationiteration, betingelser (if-else) og math-random.

Mit spil

Jeg valgte at lave et julespil, hvor man skulle klikke på de dårlige elementer for at få point. Jeg startede det hele i adobe illustrator og fik designet mine baggrunde og figurer. Derefter exportede jeg dem som svg, så det hele kunne indsættes i html og css. Alt andet blev foretaget inde i javascript og css, hvor jeg fik kodet tid, elementernes random positioner og delays, liv og point. Selvom javascript var svært, var det her nok mit favoritemne!
