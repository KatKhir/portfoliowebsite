TEMA 4

Grundlæggende animation

Om forløbet

Nu skulle det handle om dynamiske websites. I dette forløb kombinerede vi JavaScript og CSS animationer til at udvikle et spil. Vi blev introduceret for Adobe Illustrator hvori vi skulle arbejde med formgivning af grafiske spil-elementer. Vi skulle bruge nye idégenererings-teknikker, lave figur-skitser, lære om CSS-positionering og om export af figurer til SVG. Derudover skulle vi fremstille diagrammer over vores spil-forløb og kode ud fra dem. Vi fik gode introduktioner til hvordan JavaScript fungerer, herunder funktioner, metoder (addClass/removeClass), events (load, click, animationend, animationiteration), betingelser (if-else) og math-random.

Mit spil

Jeg valgte at lave et julespil, hvor man skulle klikke på de dårlige elementer for at få point. Jeg startede det hele i Adobe Illustrator og fik designet mine baggrunde og figurer. Derefter exportede jeg dem som SVG, så det hele kunne indsættes i HTML og CSS. Alt andet blev foretaget inde i JavaScript og CSS, hvor jeg fik kodet tid, elementernes random positioner og delays, liv og point og lydeffekter. Selvom JavaScript var svært, var det her nok mit favorit emne!
