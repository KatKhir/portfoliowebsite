TEMA 2

Grundlæggende web

Mangler titel

I vores første rigtige forløb blev der åbnet en værktøjskasse med de mest anvendte redkskaber man som multimediedesigner gør brug af. Vi blev præsenteret for kodningsprogrammet Visual Studio Code og lærte at sætte websider op i html og anvende css grid, flex og media queries. Derudover blev vi introduceret til nye faglige begreber som brugergrænseflader, digital kommunikation og responsivt webdesign. Vi arbejdede med wireframes, styletiles og tekstopsætning i programmet Figma, som vi også lige var blevet introduceret for.

Studiestartsprøven

Vi skulle i dette forløb beskræftige os med en masse forskellige opgaver som senere skulle implimenteres i et website som udgjorde vores studiestartsprøve. I opgaverne lærte jeg at lave responsivt design - at gå fra mobil til desktop - ved hjælp af media queries, og at få opbygget et layout ved hjælp af grid. Dette var en rigtig god metode for både at lære og udføre noget selv. Alt indholdet i dette projekt var valgt på forhånd af vores undervisere, og derfor lå vores fokus på at lave et flot design med selvalget farver.
